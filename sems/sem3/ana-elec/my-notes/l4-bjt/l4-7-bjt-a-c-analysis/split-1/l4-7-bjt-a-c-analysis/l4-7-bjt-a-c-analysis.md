# **Bipolar Junction Transistor**

#### **BJT AC Circuit Analysis**

- AC equivalent of a transistor is obtained by:
  - 1. Setting all DC sources to zero.
  - 2. Replacing all capacitors to short circuit equivalent.
  - 3. Removing all elements bypassed by short equivalent by steps 1 and 2.
  - 4. Redrawing the network in more convenient and logical form.
